# Log Analyzer

Um analisador de logs desenvolvido em Python para identificar eventos de interesse em arquivos de log.

## Objetivo

Este projeto foi criado com fins de estudo e para compor meu portfólio de Segurança da Informação, simulando uma atividade de busca de ocorrências específicas.

## Funcionalidades

- Lê um arquivo de log.
- Percorre todas as linhas do arquivo.
- Identifica ocorrências.
- Exibe as linhas que contêm erros.
- Informa a quantidade .
